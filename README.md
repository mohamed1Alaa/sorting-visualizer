# sorting-visualizer

Sorting Visualizer is a web app for visualizing a bunch of different sorting algorithms Like Selection
Sort,Bubble Sort,Insertion Sort,Merge Sort,Quick Sort With the functionality of Speed Control and
Array Size Control.
